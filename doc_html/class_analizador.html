Descripción detallada

Clase que estudia los datos de un objeto imagen, mapea superficies en la imagen y define las funciones del histograma.

Esta clase realiza cálculos de promedio, desvío estándar, moda, mínimo y máximo valor y máxima aparición de los valores de píxeles en la imagen. Define las curvas de las funciones del histograma. Es capaz de mapear superficies mediante algoritmo del pintor.
