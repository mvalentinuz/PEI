Descripción detallada

Clase que hereda de Filtro y opera sobre los píxeles de una imagen aplicando un efecto normalización de valores RGB.

Esta clase filtra una imagen mediante el estudio de píxeles vecinos para establecer un valor medio y quitar el efecto de sal y pimienta.
