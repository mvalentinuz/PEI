Descripción detallada

Clase que se comunica con el usuario indicando funciones de uso. Coordina las operaciones de los objetos fundamentales ante las entradas del usuario.

Esta clase expone por consola un menú de opciones y recibe instrucciones del usuario para ejecutar las responsabilidades del objeto expositor.
